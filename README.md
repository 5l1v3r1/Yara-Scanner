# Yara-Scanner

## Introduction
Python-based Yara scanner intergrated into Burp Suite. Yara-Scanner allows you perform on-demand scans of websites within the Burp interface, based on custom Yara rules that you write. Tested in Burp Suite Pro 1.6.3x. 

## Prerequisite: Jython
If you have not already added Jython to Burp:
1. Download the latest version of Jython (2.7) from http://www.jython.org/downloads.html
2. In Burp, go to the Extender tab, then Options
3. Under Python 

## How to Install:
1. Download yaraburp.py
3. In Burp, go to the Extender tab, then Extensions
3. Click Add
4. Under Extension details, set Extension type to Python
